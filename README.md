SPACE GAME
==========

This project is a game inspired by the classic Space Invaders. A spaceship fights waves of enemy ships.

The purpose of this project was to have fun with an AI code generator. The prompts (majority, not all) can be found in the prompts.md file.
Rust was chosen because I do not have much experience with it, so I used this as an opportunity to learn how the code might look.

The graphics of the player's ship, enemy ships, and a bomb were generated by Google Gemini and edited in GIMP to add a transparent background via the alpha channel.

# Controls & Rules

Use the L & R cursor keys to move the player's ship left and right. Avoid diving enemy ships and dropped bombs. The player has 3 lives.

Press SPACE to shoot laser beams and destroy enemies. Watch out! The gun overheats when fired too often. Once overheated, the player must wait for it to cool down.

Shooting down an enemy ship gives 1 point. Destroying a diver gives 5 points.

After clearing a wave of ships, another wave will come.

With each wave, the enemy ships move faster and divers appear more frequently.

Press R to restart the game.
